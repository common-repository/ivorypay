== Description ==
Ivorypay is a payment gateway that enables African businesses to accept payments in stable cryptocurrencies such as USDT and USDC. African businesses that integrate with Ivorypay can process both local and international payments and withdraw their funds in their local fiat currency. Ivorypay’s plugin is available to businesses in Nigeria, Ghana, Kenya, and South Africa.

With Ivorypay, you can accept payments via the following options;

Payment button and checkout
Payment link
QR code

= Why Choose Ivorypay? =

* Instant signup. Start accepting stable crypto payments within 15 minutes of signing up
* Expand Payment Options: Offer more convenient payment options to both your local and international customers who own stable cryptocurrencies (over 53 million Africans own cryptocurrencies).
* Transparent Pricing: Enjoy a flat fee of 1% on all transactions (50% less than traditional payment gateways). 
* Ivory Storefront: Build custom online shops for your business with a few clicks.
* Quick Payouts: Withdraw into your local bank accounts in less than 1 hour (It takes 24 hours for traditional payment gateways)
* Seamless Checkout: Provide a seamless checkout and scan-to-pay experience for your customers.
* Interactive Dashboard: Get an interactive dashboard for better insights into business
* Simple API Documentation: Use clearly documented API to easily integrate.
* 24/7 Customer Support: Get quick and responsive assistance from Ivorypay’s customer support team 24/7 

Ivorypay’s product suite not only allows businesses of all sizes to accept payments in stable cryptos anywhere and at any time, but it’s also built for your business to scale.

Sign up on [Ivorypay](https://dashboard.ivorypay.io/register) to get started

== Installation ==
= How to install Ivorypay WordPress Plugin =
* On your WordPress dashboard, click plugins. Then, add new plugin
* Search for “Ivorypay” in the search bar located at the right-hand side of your screen.
* Click install plugin. Once the plugin is installed, click Activate.
* From here, go to installed plugins. Click on configure to edit the details of the plugin
* Input your Ivorypay PUBLIC API key in the AUTHORISATION field. Note that only merchants that are registered on Ivorypay can access the Public Key. 
* Then save changes.

= Important notice =
* To obtain your API Keys on Ivorypay Dashboard, follow the steps below:
1. Log into your Ivorypay account
2. Go to settings and select on the API and Webhooks tab
3. Your API keys will available on the page

* You can switch between LIVE and TEST mode by clicking on the toggle located at the top right corner of your screen.
* To test in TEST mode, use the TEST PUBLIC KEY. For LIVE mode testing, use the LIVE PUBLIC KEY.
* To get LIVE mode API public key, you need to submit KYC documents for approval. Read how to complete your KYC verification [here](https://www.ivorypay.io/blog/how-to-complete-kyc-for-your-business-on-ivorypay)
* After entering the public key and saving the changes, you can start accepting payments in stable cryptocurrency on your WordPress website.

To avoid loss of funds, do not use real tokens when testing with your TEST mode public keys.

== Frequently Asked Questions ==
= What do I need to use the Ivorypay plugin? =
* You will need to have a functional business account on Ivorypay in order to use this plugin. This could be an Ivorypay starter account or a registered account. Get started by creating an account [here](https://dashboard.ivorypay.io/register)
* An active [WooCommerce installation](https://woocommerce.com/document/installing-uninstalling-woocommerce/)
* A valid [SSL certificate](https://woocommerce.com/document/ssl-and-https/)
